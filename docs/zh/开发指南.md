# 开发指南

## 技术栈

- 前端：SvelteKit + TypeScript
- 桌面容器：Tauri 2
- 后端：Rust
- 存储：SQLite（元数据）+ 本地图片文件

## 本地运行

```bash
pnpm install
pnpm dev
```

## 核心模块

- `clipboard_watcher.rs`：300ms 轮询剪贴板变更，记录文本/图片。
- `storage.rs`：本地持久化、去重、限量清理。
- `permissions.rs`：辅助功能权限检测与系统设置引导。
- `selection.rs`：选中记录后的复制/自动粘贴策略。
- `commands.rs`：前后端命令契约。

## 测试

```bash
pnpm test
```

## 调试建议

- 若自动粘贴失败，优先检查辅助功能权限。
- 若列表不更新，先确认应用未被列入排除来源。
